## 交易哲學
- **穩定優先於刺激**，可重複性比高報酬更重要
- 只做 A 級以上設置（A+ / A），不做 B 級
- 趨勢優先：方向必須與高時間框架一致
- **先計畫交易，再根據計畫交易**，不隨意猜想
- 關鍵價位全部透過 Python 程式從數據運算，不靠感覺

## 交易風格
- **混合型**：順勢突破（S1）+ 關鍵位反轉（S2）
- 不追價，等回調至確信位置再進
- 目前策略方向：**Long Only（只做多）**
- 年化報酬目標：20-30%，低回撤

## 進場條件

### A+ 區（主戰場／防守區）
- 極限回調區，通常結合 4H / Daily BB 中軌
- 前壓力轉支撐（R→S）結構
- 最高優先，部位可加大

### A 區（次優，需確認 30m 結構）
- 趨勢中的「起漲點」或「缺口回補區（FVG）」
- 需看 30m K 棒結構確認再進

### S1 進場訊號三要素
1. 價格回調至關鍵位
2. 30m K 棒收下影線（錘頭型態）
3. AO 動能轉強（翻正）

### 進場等級
- S1 / S2：A+ 和 A 都做，**不做 B 級**

## 風險管理
- 每筆風險：總資金 **1.0 ~ 1.5%**
- 同時持倉：**最多 1 筆**
- 每週最多交易：**5 次**
- 利潤目標：**1R ~ 2R**
- 不加碼、不攤平、不情緒性平倉
- 強烈拒絕超大 SL 或情緒性出場

## 分析架構（每次分析必須涵蓋）

### 大格局（Context）
- 先看前三週趨勢表（W-3, W-2, W-1）
- 確認目前是「主升」、「主跌」還是「盤整」

### 波動率
- 評估目前漲跌幅是否在日均震幅合理範圍（ATR）

### 技術濾鏡
- BB（布林通道）：位置、寬度、%B
- AO（Awesome Oscillator）：方向、翻正/翻負
- 1H / 4H EMA 斜率
- DXY 趨勢：美元強弱對黃金的影響方向

### 位置濾鏡（關鍵）
- 支撐阻力（SNR）
- Fair Value Gap（FVG / 缺口）
- 溢價／折價區間（Premium / Discount）

### Session 濾鏡
- 亞/歐/美盤切換
- 星期行為分析（週幾的勝率特性）

## 每週工作流程

### 週日（週末周報）
1. 回顧前三週走向
2. 搭配布林通道 + S1-AweWithBB
3. 找出下週 A+ 與 A 的關鍵價位
4. 確認 DXY 趨勢方向
5. 輸出周報（劇本 A / B / C + 策略執行計畫）

### 週三（週間更新）
1. 根據週日周報進行週四五調整
2. 預測週線如何收線
3. 確認是否有 A+ 機會
4. 若有重大波動，加開緊急週報

### 日常（TradingView 跳訊號時）
1. 手機收到 TV 通知
2. 開 Claude co-work → 請分析
3. 對比周報關鍵價位
4. 決定是否進場
